# 项目计划

## 需求
1、创建一个ParkCraft项目，要求Android和iOS版本都支持。
2、项目采用Flutter框架开发，要求在Flutter 2.0及以上版本上运行。
3、项目采用Material Design设计规范，要求在Android和iOS版本上都有良好的用户体验。
4、项目采用Dart语言开发，要求在Dart 2.0及以上版本上运行。
5、是一款游戏，模拟停车出库的游戏，有一辆车被卡在了一个NxN的网格中，玩家需要通过移动车辆，将车辆从网格唯一的出口退出，退出口固定，待移动车辆的所在行列也固定。
6、游戏中有若干车辆堵塞在目标车辆的前面或者后面，车辆只能前后移动，待出库车辆长度为2，其余车辆有的为2，有的为3。
7、关卡通过配置文件生成，配置文件包含网格大小、车辆数量、车辆长度、车辆位置、出口位置等信息。支持快速导入。
